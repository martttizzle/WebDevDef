    Chapter 13: CHANGE STYLING INLINE

    Directly change or manipulate styling.

    $("h2.important").css("color", "red");

    Or you can change multiple properties:

    $("h2.important").css({
    "font-family": "'Times New Roman', Times, serif",
    "font-size": "1.3em",
    "color": "red"
    }):

    Chapter 14: THIS

    Inline Method to change text color:

    $("p.color_changeable").on("mouseover", function() {
    $("p.color_changeagle").css("color". "black");
    });

    If you want single element changed use keyword "THIS"

    $("p.color_changeable").on("mouseover", function() {
    $("this").css("color", "black");
    });

    Chapter 15: SELECT MULTIPLE ELEMENTS

    You can target as many elements as you want in a single statement.

    $("h3, p.not_so_important, div#fine_print,
    ul#things_to_check li.secondary").css("color", "#999");

    Chapter 16: AN ALTERNATIVE FOR SELECTING MULTIPLE ELEMENTS

    By using the .add the code will be faster for the processor but not you. The first
    target--h3--is handled normally. Additonal targets are handled with add("[selector]")

    $("h3").add("p.not_so_important").add("div#fine_print) .add("ul#things_to_check").css("color", "#999");

    Chapter 17: REPLACE AN ID

    Replace an "id" or "class" with another one, using the attr() method

    $("div#d1").on("mouseover", function() {
    $(this).attr("id", "d2");
    });

    Chapter 18: REMOVE THINGS

    You can remove all elements or leave the elements and remove contents with remove() and empty()
    example:

    $("img").remove(); OR
    $("p.expendable").remove();

    To leave the opening and closing tags but delete the content:

    $("p.customizable").empty();


    You can fill empty paragraphs with new content using text() or html() methods

    $("h3#greeting").text("Hi");

    Remove all paragraphs and ordered lists including their tags.

    $("p").add("ol").remove();

    Chapter 19: TOGGLE A CLASS

    The toggleClass() method adds the class if the element doesnt have the class and removes the class if the element does have the class.
    example:

    button> id="toggle_images"> HIDE/UNHIDE_IMAGES /button>

    .invisible {
    display:none;
    }

    $("button#toggle_images").on("Click", function() {
    $(img).toggleClass("invisible");
    });

    Toggle the class "fade" for all paragraphs whose class is "extra" and all images whose class is "extra".

    $("p.extra, img.extra").toggleClass("fade");

    When the user mouses over a paragraph, a class toggles for that paragraph. When the user mouses out, it's toggled again. Make up the paragraph ID and the class name.

    $("p#graph3").on("mouseover", function() {
    $(this).toggleClass("big");
    });
    $("p#graph3").on("mouseout", function() {
    $(this).toggleClass("big");
    });

    Chapter 21: ANOTHER WAY TO REPLACE

    You can select elements by their contents and target any headings that contain the text you want. The CSS selector name is wrapped in quotation mars, and then in parantheses. Instead of a # that would select an ID, you write a colon ":", then keyword Contains

    $("h3:contains('Hello,Dude')");

    Paragraphs contain "hazard" replaced by class"bold".

    $("p:contains('hazard')").attr("class", "bold");

    Paragraphs contain "hazard" replace their class with "bold".

    $("p:contains('hazard')").attr("class", "bold");

    Chapter 22: VARIABLES IN JQUERY

    You can store the CSS selector into a variable.

    var tgt = "p#slider"; var slide_speed = 2500;
    $(tgt).hide();
    $(tgt).slideDown(slide_speed);
    $("body").on("click", function() {
    $(tgt).slideUp();
    });

    Chapter 23 :VARIABLES NAMES LEGAL AND ILLEGAL

    You can name variables anthying you like as long as you follow javascript's variable-naming rules.
    • Can't be enclosed in quotation marks.
    • Can't contain space.
    • Can contain only letters, numbers, dollar signs, and underscores.
    • Can't start with a number.
    • Use camelCase when name variables.
