    A SMARTER WAY TO LEARN JQUERY

    The content for learning the critical aspects of jQuery are derived from the book written by Mark Myers called: "A Smarter Way to Learn jQuery." This section breaks down the main points of content of each chapter along with description of code. No code is shown for some chapters that do not require a demonstration.

    Chapter 11: SLIDE SOMETHING INTO VIEW

    You can make an image gradually appear by expanding from nothing to full size. In CSS, the image is styled so it's initially hidden.

    img#dramatic {
    display: none;
    }

    An Html tag puts the invisble phantom on the page, along with a button:

    <img src="image.jpg" width ="200" id="dramatic"
    height="218" alt="imag2">
    <button id='make_visible">Show the image </button>

    When the button is clicked jQuery slides the image into view using slideDown() :

    $("button#make_visible").on("click", function() {
    $("img#dramatic").slideDown();
    });

    You can also use slideUp() to make it out of view and include time in milliseconds.

    $("img#dramatic").slideUp(5000);

    Chapter 12: SHOW AND TOGGLE

    You can also show and hide things instantly. .show() instantly displays a div(including all its contents) that has been hidden:

    $("div#onceHidden").show();

    The following code toggle() switches back and forth. It makes instantly visible all ordered lists of a certain class
    if they're hidden - and hides them if they're visible:

    $("ol.switched").toggle();

    You can add "slow", "fast", or enter a number to determine the speed of show(), hide(), and toggle().

    $("div#d3").toggle(3000);

    Chapter 13: CHANGE STYLING INLINE

    Directly change or manipulate styling using css("color", "red ") .

    $("h2.important").css("color", "red");

    Or you can change multiple properties:

    $("h2.important").css({
    "font-family": "'Times New Roman', Times, serif",
    "font-size": "1.3em",
    "color": "red"
    }):

    Chapter 14: THIS

    Inline Method to change text color:

    $("p.color_changeable").on("mouseover", function() {
    $("p.color_changeagle").css("color". "black");
    });

    If you want single element changed use keyword "THIS" .

    $("p.color_changeable").on("mouseover", function() {
    $("this").css("color", "black");
    });

    Chapter 15: SELECT MULTIPLE ELEMENTS

    You can target as many elements as you want in a single statement.

    $("h3, p.not_so_important, div#fine_print,
    ul#things_to_check li.secondary").css("color", "#999");

    Chapter 16: AN ALTERNATIVE FOR SELECTING MULTIPLE ELEMENTS

    By using the .add the code will be faster for the processor but not you. The first
    target--h3--is handled normally. Additonal targets are handled with add("[selector]")

    $("h3").add("p.not_so_important").add("div#fine_print) .add("ul#things_to_check").css("color", "#999");

    Chapter 17: REPLACE AN ID

    Replace an "id" or "class" with another one, using the attr( ) method

    $("div#d1").on("mouseover", function() {
    $(this).attr("id", "d2");
    });

    Chapter 18: REMOVE THINGS

    You can remove all elements or leave the elements and remove contents with remove() and empty()
    example:

    $("img").remove(); OR
    $("p.expendable").remove();

    To leave the opening and closing tags but delete the content:

    $("p.customizable").empty();


    You can fill empty paragraphs with new content using text( ) or html( ) methods

    $("h3#greeting").text("Hi");

    Remove all paragraphs and ordered lists including their tags.

    $("p").add("ol").remove();

    Chapter 19: TOGGLE A CLASS

    The toggleClass( ) method adds the class if the element doesnt have the class and removes the class if the element does have the class.
    example:

    <button id="toggle_images"> HIDE/UNHIDE_IMAGES </button>

    .invisible {
    display:none;
    }

    $("button#toggle_images").on("Click", function() {
    $("img").toggleClass("invisible");
    });

    Toggle the class "fade" for all paragraphs whose class is "extra" and all images whose class is "extra".

    $("p.extra, img.extra").toggleClass("fade");

    When the user mouses over a paragraph, a class toggles for that paragraph. When the user mouses out, it's toggled again. Make up the paragraph ID and the class name.

    $("p#graph3").on("mouseover", function() {
    $(this).toggleClass("big");
    });
    $("p#graph3").on("mouseout", function() {
    $(this).toggleClass("big");
    });

    Chapter 20: REPLACE THINGS

    To completely remove all elements and all on contents(blank) you use replaceWith( ). You type in the tags and id's and etc...for the replacement text.
    example:

    $("p #greeting").replaceWith("<p> id='greeting'
    class='austrailian'><em>G'day,Mate.</em></p>

    Chapter 21: ANOTHER WAY TO REPLACE

    You can select elements by their contents and target any headings that contain the text you want. The CSS selector name is wrapped in quotation mars, and then in parantheses. Instead of a # that would select an ID, you write a colon ":", then keyword Contains

    $("h3:contains('Hello,Dude')");

    Paragraphs contain "hazard" replaced by class"bold".

    $("p:contains('hazard')").attr("class", "bold");

    Paragraphs contain "hazard" replace their class with "bold".

    $("p:contains('hazard')").attr("class", "bold");

    Chapter 22: VARIABLES IN JQUERY

    You can store the CSS selector into a variable.

    var tgt = "p#slider"; var slide_speed = 2500;
    $(tgt).hide();
    $(tgt).slideDown(slide_speed);
    $("body").on("click", function() {
    $(tgt).slideUp();
    });

    Chapter 23: VARIABLES NAMES LEGAL AND ILLEGAL

    You can name variables anthying you like as long as you follow javascript's variable-naming rules.
    • Can't be enclosed in quotation marks.
    • Can't contain space.
    • Can contain only letters, numbers, dollar signs, and underscores.
    • Can't start with a number.
    • Use camelCase when name variables.

    Chapter 24: PERSONALIZE A HEADING

    You can use store input's into a variable and use it to for other headings:

    $("form#userName").on("submit", function() {
    var first_name = $("input#firstName").val();
    var last_name = $("input#lastName").val();
    $("h4#personalizedQuote").text("Personal Quote for " + first_Name + " " + last_name);
    });

    Chapter 25: ADD EXTISTING HTML TEXT

    If you want to bold some text in response to a user's action. do it by combining two methods you already know: html() & text() .

    var okToBold = $("p#important");

    ...add the html() method...

    okToBold.html(""<p id='important'><strong>" + okToBold.text() +"</strong></p>");

    Example of storing the text of a paragraph into a variable.

    var a = $("p#id").text();

    Another example if the user clicked a paragraph it bolded the paragraph.

    $(this).html("<p><strong>" + $(this).text() + "</strong></p>");

    Or another example would be when the user clicks out of a field, fill the field with a concatenation of "Ms. " plus the contents of the field.

    $("input#id").on("blur", function() {
    $(this).val("Ms. " + $(this).val());
    });

    Chapter 26: CHAINING

    You can chain methods together using the dot operator . between methods.

    $("div#main").css("color", "black").css("font-size", "1.5em");

    Another example with graph as a variable:

    graph.toggleClass("prominent").css("font-weight", "bold");

    Another example:

    $("input#species").attr("id", "default").val("human");
